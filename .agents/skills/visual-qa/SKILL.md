# Visual QA Skill

Use this skill when reviewing screenshots for a visual change.

## Scope

Review the specific section or pattern changed. Do not require full-page review unless the change affects the whole page.

## Required Screenshots

- desktop
- tablet
- mobile

## Review Criteria

- spacing
- alignment
- overflow
- text readability
- CTA visibility
- visual hierarchy
- image cropping
- mobile stacking
- tablet behavior
- desktop behavior
- consistency with `DESIGN_SYSTEM.md`

## Report Format

Include:

- screenshots reviewed
- issues found
- severity
- recommended fixes
- approval status
